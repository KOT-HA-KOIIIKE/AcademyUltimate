Сервис проверяет корректность верстки статей через HTML. Сервис доступен по IP.

Проверка опубликованных статей: вставить ссылку -> нажать "Проверить".

Проверка еще неопубликованных статей: открыть предпросмотр черновика -> ПКМ -> "Сохранить как" -> HTML -> "Сохранить" -> "Выберите файл" -> "Проверить".

Апдейт после коммита: sudo pkill gunicorn -> git pull -> sudo ./venv/bin/gunicorn --bind 0.0.0.0:80 --timeout 120 app:app --daemon
